# CocoPass


CocoPass is a password management system based on Laravel5.3. You can safely and easily store your password here.
## Installation
Option 1: 

  ```bash
  composer install
  ```

### Option 2: create a .env file and write MySql info in it

  ```bash
 touch .env
  ```
###Option 3: migrate 
  
  ```bash
 php artisan migrate
  ```
